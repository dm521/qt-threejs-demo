# 模型文件目录

**注意：GLB文件应放置在项目根目录下，而不是此目录中。**

## 文件结构
```
threejs-animation-demo/
├── background.glb      # 您的背景场景GLB文件（放在根目录）
├── models/
│   └── README.md      # 此说明文件
└── ...
```

## 支持的格式
- GLB (.glb)
- GLTF (.gltf)

## 压缩支持
- **DRACO压缩**: 支持DRACO几何体压缩，可显著减少文件大小
- **自动解码**: 使用Google CDN的DRACO解码器

## 注意事项
- 如果GLB文件不存在，系统会自动使用占位符模型
- 确保模型文件大小合理，避免影响加载性能
- 建议模型包含适当的材质和纹理
- DRACO压缩的模型加载更快，文件更小
